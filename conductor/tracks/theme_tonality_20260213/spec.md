# Specification: Система тем и тональностей общения

## Описание
Реализация гибкой настройки внешнего вида и языка интерфейса для Telegram Mini App "Сводный Вестник Скидок". Система должна позволять пользователю выбирать один из трех визуальных стилей и один из трех стилей общения.

## Функциональные требования
1. **Хранение настроек**: Сохранение выбора пользователя в Pinia (frontend) и синхронизация с `localStorage`.
2. **Визуальные стили**:
    - **Glassmorphism**: Основной стиль с `backdrop-filter: blur`, тенями и прозрачностью.
    - **Pure Android**: Плоский дизайн Material 3, динамические акценты.
    - **Liquid Glass**: Стиль iOS с экстремальными скруглениями и системными шрифтами.
3. **Тональность общения (Tone of Voice)**:
    - **Ироничный/Торжественный**: Дореволюционный стиль (Милостивый государь, извольте).
    - **Дружелюбный**: Современный нейтральный язык.
    - **Хайповый**: Сленг и эмодзи (Чекай прайс, сейл).
4. **Интеграция**: Переключатели в `SettingsModal.vue`.

## Технические детали
- Использование CSS-переменных для переключения визуальных тем.
- Создание объекта-словаря (или хука) для перевода строк интерфейса в зависимости от выбранного тона.
- Поддержка "Educational Bad UX" (эффект размытия при наведении) во всех темах.
